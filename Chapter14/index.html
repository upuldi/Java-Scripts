Objects

You can create object in JS using new Object() or simply using {}. Properties can be created with [] or usig dot operator.

There are two ways of creating properties in JS.

Using computed memeber access operator - ([])

[] can be used to define properties and set values to objects. eg : object['property'] = XXX and object['property'].

Using memeber access operator - DOT (.)

{} is an object literal, meaning you can use it to create objects without using new
